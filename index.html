Tentang Kita

PT KAI Indonesia, juga dikenal sebagai Kereta Api Indonesia, adalah perusahaan yang beroperasi di sektor transportasi kereta api di Indonesia. Perusahaan ini didirikan pada tahun 1945 dan sejak itu telah menjadi penyedia layanan kereta api terbesar di Indonesia.
